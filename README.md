# c++ timer

A timer alarm that runs in the terminal. 

To compile:

* make clean
* make ( for terminal run )
* make mac ( creates a .app file in root directory)
* ( no win app instructions was added to the makefile sorry )

To run:

* either run ./TimerApp
* or click TimerApp in root directory
